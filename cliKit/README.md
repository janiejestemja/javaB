# cliKit
---
Commandline toolkit for Java development.

## Quickstart
---
Enable bash script
```bash
chmod +x cliKit.sh
```

### Usage
---
Compile Package
```bash
./cliKit.sh -p <packageName>
```
Compile Main
```bash
./cliKit.sh -c
```
Execute Main
```bash
./cliKit.sh -e
```
Compile and Execute Main
```bash
./cliKit.sh -ce
```
